# Install

```console
$> git clone git@github.com:o2t/tutorial-node-stock-server.git
$> yarn
```

# Run

```console
$> yarn start
```

# Query

To load the last 10 stock value samples :

```console
$> curl http://localhost:8000?count=10
```

To load all available stock value samples :

```console
$> curl http://localhost:8000
```
